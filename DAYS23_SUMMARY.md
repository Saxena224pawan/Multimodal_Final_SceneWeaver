## Architecture & Integration Points

```
Pipeline Wrapper
├─ Load existing models (video, director, embedder, captioner)
├─ Initialize MultiAgentIntegrator
├─ For each window:
│  ├─ Generate base prompt
│  ├─ Run RefinementEngine (up to 3 iterations)
│  │  ├─ Generate video
│  │  ├─ Run 3 agents in parallel
│  │  ├─ Score & aggregate
│  │  ├─ If score < 0.70 → Optimize prompt & retry
│  ├─ Save metadata (JSON)
│  ├─ Continue with next window
├─ Save convergence summary
└─ Generate evaluation reports

Analysis Pipeline
├─ Load all metadata files
├─ Compute convergence statistics
├─ Run ablation studies
├─ Generate reports (JSON)
├─ Create visualizations (PNG)
└─ Output publication-ready figures
```

## Ready-to-Use Workflows

### Workflow 1: Generate with Multi-Agent Refinement
```bash
python scripts/run_story_pipeline_with_agents.py \
    --storyline "Your story text or file.txt" \
    --output-dir outputs/my_run \
    --max-iterations 3 \
    --quality-threshold 0.70
```

### Workflow 2: Evaluate Results
```bash
python evaluation.py outputs/my_run
```

- `convergence_report.json` - Statistics on iterations & scores

## Technical Details

### Components Integrated
1. **Video Model** (WanBackbone) - Frame generation
2. **Scene Director** (LLM) - Story planning
3. **Captioner** (BLIP-2/LLaVA) - Frame descriptions
4. **Embedder** (CLIP/DINOv2) - Visual consistency
5. **Multi-Agent System** - Evaluation & refinement

### Pipeline Stages
1. Story planning → Windows with beats
2. For each window:
   - Generate prompt
   - Generate frames (with potential refinement)
   - Extract captions
   - Run agent evaluation
   - Aggregate scores
   - Optional: Refine & regenerate
3. Collect metrics and save results
4. Analyze convergence & generate reports
5. Create visualizations

### Metrics Tracked
- **Per-window**:
  - Total iterations to acceptance
  - Score history (per iteration)
  - Agent scores (continuity, storybeats, physics)
  - Generation time
  - Character names, location, beat text

- **Aggregate**:
  - Convergence statistics
  - Agent importance (ablation)
  - Score distributions
  - Efficiency metrics
